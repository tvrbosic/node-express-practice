## Mongo DB

- Running locally on Linux: `sudo systemctl mongod`

- Manage database using application: `MongoDB Compass`

- Manage database using shell by running: `mongosh`.

- Official documentation: [MongoDB Manual](https://www.mongodb.com/docs/manual/)
